# vimfiles

My `~/.vim` and `~/.vimrc`.

# Installation

```sh
git clone --recursive https://github.com/caarlos0/vimfiles.git ~/.vimfiles
ln -s ~/.vimfiles/vim.symlink ~/.vim
ln -s ~/.vimfiles/vimrc.symlink ~/.vimrc
vim +PluginInstall +qall
```
